# 🕷️ Black Widow — Natasha Romanoff Fan Website

A cinematic, fully responsive fan website dedicated to **Natasha Romanoff / Black Widow** from the Marvel Cinematic Universe. Built with pure HTML, CSS, and vanilla JavaScript — no frameworks, no dependencies, just open and run.

---

## 📁 Project Structure


---

## 🚀 How to Run

1. **Extract** the ZIP file
2. Open the `blackwidow/` folder
3. Double-click **`index.html`**
4. Opens directly in any modern browser — no server needed

> ✅ Works in Chrome, Firefox, Edge, Safari  
> ✅ No installation, no Node.js, no build step

## 🌐 Sections

| Section | Description |
|---|---|
| **Hero** | Full-screen cinematic intro with animated title and background |
| **Sliding Banner** | Auto-scrolling image strip — 10 unique photos, pausable on hover |
| **Story** | Natasha's origin — Red Room, S.H.I.E.L.D., Avengers |
| **Chronicle** | 5 key MCU moments with photo + text alternating layout |
| **Quote Banner** | Highlighted quote from Age of Ultron |
| **Timeline** | Full MCU appearance timeline from 2010–2021 |
| **Abilities** | 6 skill cards with animated progress bars |
| **Photo Gallery** | All 20 images in a CSS masonry grid — zero cropping |
| **Videos** | All 4 videos with native controls — full size, no cropping |
| **Contact** | Contact form + social/professional links |
| **Footer** | Navigation links and credits |

---

## ✨ Features

- **Custom animated cursor** — red dot + ring that follows the mouse
- **Floating particles** — subtle red particle animation in background
- **Scroll animations** — elements fade and slide in as you scroll
- **CSS Masonry gallery** — photos display at natural proportions, no cropping
- **Auto-sliding image strip** — pauses on hover
- **Spider logo divider** — animated SVG between sections
- **Hero background pan** — subtle Ken Burns effect on hero image
- **Scanline overlay** — cinematic CRT texture over hero
- **Animated skill bars** — trigger when scrolled into view
- **Glowing red accents** — consistent red/dark theme throughout
- **Mobile responsive** — hamburger nav, single-column layouts on small screens
